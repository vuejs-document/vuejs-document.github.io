安装

兼容性提示: Vue.js不在支持IE8及以下版本

独立版本

简单的下载并用script标签插入到页面上，Vue将会被注册为全局变量。


开发环境版本带有全部的警告信息并且是debug模式

生产环境版本删掉警告的 23.66kb min+gzip

CDN

jsdelivr 或 cdnjs都是可用的 (需要一点时间去同步最新版本，因此最新版本可能不可用)

CSP兼容版本

有一些环境，比如在Google Chrome Apps里，会严格执行内容安全政策(CSP)，因此不允许使用new Function()来进行表达式求值，在这种情况下，你可以使用CSP兼容版本代替。

NPM

当使用Vue.js创建大型的应用程序的时候推荐使用NPM安装方式。Vue.js也可以很好的跟CommonJS模块打包器一起配合使用，比如Webpack 或 Browserify。Vue.js也提供相关的工具Single File Components。

Bower

AMD 模块加载器

独立下载或者通过Bower下载指定版本，都是使用UMD方式处理的，所以他们可以直接作为AMD模块使用。
